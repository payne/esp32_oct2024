# Introduction to ESP32 for DIY Electronics Hobbyists

---

## What is ESP32?

- Powerful, low-cost microcontroller
- Built-in Wi-Fi and Bluetooth
- Perfect for IoT projects
- Developed by [Espressif Systems](https://www.espressif.com/)

---

## What to Buy: Getting Started Kit

1. ESP32 development board (e.g., ESP32-WROOM-32)
2. Micro-USB cable
3. Breadboard
4. Jumper wires
5. Basic components (LEDs, resistors, buttons)
6. Optional: sensors (temperature, humidity, motion)

---

## Development Environments

1. [Arduino IDE](https://www.arduino.cc/en/software)
   - Easy to use for beginners
   - Extensive library support

2. [ESP-IDF](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/) (Espressif IoT Development Framework)
   - More advanced, full-featured
   - Better performance and control

3. [MicroPython](https://micropython.org/)
   - Python-based programming
   - Great for rapid prototyping

---

## Setting Up Arduino IDE for ESP32

1. Install [Arduino IDE](https://www.arduino.cc/en/software)
2. Add ESP32 board manager URL in Preferences
3. Install ESP32 board package
4. Select your ESP32 board model
5. Choose the correct port

---

## Free Web-Based IDEs for ESP32

1. [ESP32.io](https://esp32.io/)
   - Online Arduino IDE
   - Supports ESP32 and other Arduino-compatible boards
   - No installation required

2. [PlatformIO IDE for VSCode](https://platformio.org/platformio-ide) (with [Gitpod](https://gitpod.io/))
   - Full-featured IDE in the cloud
   - Supports multiple frameworks (Arduino, ESP-IDF)
   - Accessible through web browser

3. [Espruino Web IDE](https://www.espruino.com/ide/)
   - JavaScript-based programming for microcontrollers
   - Supports ESP32 and other boards
   - Live code execution and debugging

4. [MicroPython Online IDE](https://micropython.org/unicorn/) (Unicorn)
   - Browser-based MicroPython emulator
   - Great for learning and quick experiments
   - Limited to emulation, not for flashing real hardware

---

## Benefits of Web-Based IDEs

- No installation required
- Accessible from any device with a web browser
- Often include project management and version control
- Easy to share and collaborate on projects
- Regularly updated without user intervention

---

## Considerations for Web-Based IDEs

- Require a stable internet connection
- May have limitations compared to desktop IDEs
- Some features might require paid subscriptions
- Flashing real hardware may require additional steps

## Example Project: LED Blink with Web Control

Let's create a simple web-controlled LED blink project!

---

## Hardware Setup

1. Connect an LED to GPIO 2 (with a 220Î© resistor)
2. Power your ESP32 via USB

---

## Source Code (Arduino/C++)

```cpp
#include <WiFi.h>
#include <WebServer.h>

const char* ssid = "YourWiFiSSID";
const char* password = "YourWiFiPassword";

WebServer server(80);

const int led = 2;
bool ledState = LOW;

void setup() {
  Serial.begin(115200);
  pinMode(led, OUTPUT);

  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) {
    delay(1000);
    Serial.println("Connecting to WiFi...");
  }
  Serial.println(WiFi.localIP());

  server.on("/", handleRoot);
  server.on("/toggle", toggleLED);
  server.begin();
}

void loop() {
  server.handleClient();
}

void handleRoot() {
  String html = "<html><body>";
  html += "<h1>ESP32 LED Control</h1>";
  html += "<p>LED is " + String(ledState ? "ON" : "OFF") + "</p>";
  html += "<a href='/toggle'><button>Toggle LED</button></a>";
  html += "</body></html>";
  server.send(200, "text/html", html);
}

void toggleLED() {
  ledState = !ledState;
  digitalWrite(led, ledState);
  server.sendHeader("Location", "/");
  server.send(303);
}
```

## Source Code (MicroPython)

```python
import network
import socket
from machine import Pin
import time

# Wi-Fi configuration
ssid = 'YourWiFiSSID'
password = 'YourWiFiPassword'

# Connect to Wi-Fi
wlan = network.WLAN(network.STA_IF)
wlan.active(True)
wlan.connect(ssid, password)

# Wait for connection
while not wlan.isconnected():
    time.sleep(1)
print('Connected to Wi-Fi')
print('IP:', wlan.ifconfig()[0])

# Set up LED
led = Pin(2, Pin.OUT)
led_state = False

# HTML template
html = """<!DOCTYPE html>
<html>
    <body>
        <h1>ESP32 LED Control</h1>
        <p>LED is {}</p>
        <a href="/toggle"><button>Toggle LED</button></a>
    </body>
</html>
"""

# Set up web server
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.bind(('', 80))
s.listen(5)

while True:
    conn, addr = s.accept()
    request = conn.recv(1024)
    request = str(request)
    
    if request.find('/toggle') != -1:
        led_state = not led_state
        led.value(led_state)
    
    response = html.format("ON" if led_state else "OFF")
    conn.send('HTTP/1.1 200 OK\n')
    conn.send('Content-Type: text/html\n')
    conn.send('Connection: close\n\n')
    conn.sendall(response)
    conn.close()
```

## Running the Project

1. Update Wi-Fi credentials in the code
2. Upload the sketch to your ESP32
3. Open Serial Monitor to get the IP address
4. Visit the IP address in a web browser
5. Click the "Toggle LED" button to control the LED
